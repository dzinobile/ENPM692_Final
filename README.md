# ENPM692_Final
Final project for ENPM692

### Use instructions
- Populate build_orders.csv with information about upcoming builds

- Run MES build screen
```bash
python3 injection_molder.mes.py
```
- Fill in or scan operator info, batch info, build info, equipment info
- Click +1 part to add a produced part
- Use dropdown and click log defect to log a defect
- MES will continuously populate injection_molder_mes_log.csv
